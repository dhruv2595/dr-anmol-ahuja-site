## Known site issues found during scrape

- **Blog is fully broken (server-side):** all 4 blog URLs (`/blog/`, and 3 posts) return HTTP 500. Root cause confirmed via Wayback Machine snapshots (2022–2025): `Fatal error: Uncaught mysqli_sql_exception: Access denied for user 'dranmol_blog'@'localhost'` — the WordPress blog's DB credentials have been broken since at least Dec 2022. `/blog/170-2/` has no archived snapshot at all (title/content unrecoverable).
- **Testimonials page (`/testimonials.php`) also returns HTTP 500 live.** Recovered 6 verbatim quotes from a 2023-02-06 Wayback snapshot; pagination ("Next") confirms more testimonials exist beyond what's archived.
- **Lorem ipsum placeholder text is live on `/colorectal.html`** — the "About Anmol Ahuja" bio section on that page was never replaced with real copy.
- **Meta descriptions:** not present/exposed on any scraped page.

## Caveat on copy fidelity

Body copy in `pages/**/*.md` was extracted via WebFetch, which converts HTML → markdown and passes it through a summarization pass — it is **close-to-verbatim but not byte-exact** (some paraphrase, a few link URLs/alt text marked "unspecified" where not resolvable). If pixel/word-exact copy is needed for any specific page (e.g. for direct reuse), re-scrape that page with a raw HTTP fetch instead.
